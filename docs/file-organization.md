## File Purposes

### HTML Files
- **index.html** - Main landing page with hero, services overview, and testimonials
- **about.html** - John's background, credentials, philosophy, and mission
- **services.html** - Detailed service offerings with process and pricing
- **case-studies.html** - Success stories and client testimonials
- **contact.html** - Contact form and information

### Assets
- **styles/main.css** - Complete responsive styling with GreenSpring branding
- **js/main.js** - Interactive functionality and animations
- **images/** - All visual assets organized in one location

### Documentation
- **docs/project-discovery.md** - Initial planning and discovery questions
- **docs/linkedin-profile.md** - Source data for John's professional background
- **docs/Linkedin-Profile.pdf** - PDF backup of profile information
